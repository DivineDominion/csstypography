CSSTypography

Simple styling for web typography

First, include it on your page

CSSTypography is a YUI 3 Gallery module that makes it easy to implement common web typography styles on your web site. It's not out in the YUI Gallery yet, but you can pull it in from Github.

You can fork it from Github too.
